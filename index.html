2. DOM (Document Object Model)

El DOM representa la estructura de un documento HTML que JavaScript puede manipular para modificar dinámicamente el contenido, estructura y el estilo de la página.

Principales Metodos del DOM

1. Seleccionar elementos:

getElementById(): Selecciona un elemento por su atributo id.

getEelementByClassName(): Selecciona todos los elementos que tienen un clase especifica. Retorna una coleccion de elementos.

getElementByTagName(): Selecciona todos los elemenots por su nombre de etiqueta (como p, div, etc).

querySelector(): Selecciona el primer elemento que coincide con un selector CSS.

2. Modificar elementos del DOM:

innerHTML: Modifica o establece el contenido HTML dentro de un elemento.

textContent:Modifica o establece el contenido de texto puro de un elemento, sin interpretar el HTML.

setAttribute(): Modifica o establece un atributo de un elemento.

classList.add(), classList.remove(), classList.toggle(): Manipula las clases CSS de un elemento.

style Modifica el estilo CSS en linea de un elemento.

3. Crear y eliminar nodos.

createElement(): Crea un elemento HTML.

createTextNode(): Crea un nodo de texto.

appendChild(): Agrega un nuevo nodo como hijo de un elemento.

removeChild(): Elimina un nodo hijo de un elemento.

4. Eventos en el DOM.

addEventListener(): Se usa para registrar un evento en un elemento.

Alguno de los eventos mas comunes son: click, mouseover, keyup, load.

7. Viewport

El viewport es el área visible de la página web. La meta etiqueta viewport se utiliza para controlar el comportamiento de la página en dispositivos móviles. Este archivo ya incluye la meta etiqueta viewport:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Esta configuración asegura que el ancho de la página sea igual al ancho del dispositivo y que el zoom inicial sea del 100%.

Para controlar el comportamiento del viewport en dispositivos móviles, se usa una etiqueta especial en el head del documento HTML llamada meta name="viewport". Esta etiqueta permite ajustar el tamaño y la escala del viewport.

Propiedades del Meta Viewport

width: Controla el ancho del viewport. width=device-width: Establece el ancho del viewport al ancho del dispositivo.

height: Controla la altura del viewport. height=device-height: Establece la altura del viewport al alto del dispositivo.

initial-scale: Define el nivel de zoom inicial de la página. initial-scale=1.0: No hay zoom inicial.

minimum-scale: Establece el nivel mínimo de zoom que se permite. minimum-scale=1.0: El usuario no puede hacer zoom menos de 100%.

maximum-scale: Establece el nivel máximo de zoom que se permite. maximum-scale=2.0: El usuario puede hacer zoom hasta el 200%.

user-scalable: Permite o desactiva el zoom por parte del usuario. user-scalable=yes: Permite el zoom. user-scalable=no: Desactiva el zoom.

8. Media Queries

Las media queries se usan junto con el meta viewport para crear diseños adaptativos que responden a diferentes tamaños de pantalla y dispositivos.

Propiedades en Media Queries

width y height:

  • min-width: Aplica los estilos si el ancho del viewport es mayor o igual a un valor específico.
  • max-width: Aplica los estilos si el ancho del viewport es menor o igual a un valor específico.
  • min-height: Aplica los estilos si la altura del viewport es mayor o igual a un valor específico.
  • mmax-height: Aplica los estilos si la altura del viewport es menor o igual a un valor específico.

    • orientation:

  • portrait: Aplica los estilos si el dispositivo está en orientación vertical.
  • landscape: Aplica los estilos si el dispositivo está en orientación horizontal.

    • resolution:

  • min-resolution: Aplica los estilos si la resolución es mayor o igual a un valor específico.
  • max-resolution: Aplica los estilos si la resolución es menor o igual a un valor específico.

    • aspect-radio:

  • min-aspect-ratio: Aplica los estilos si la relación de aspecto es mayor o igual a un valor específico.
  • max-aspect-ratio: Aplica los estilos si la relación de aspecto es menor o igual a un valor específico.

    • Ejemplo: Si el ancho de la pantalla es menor a 600px, el fondo será azul. Si está entre 601px y 1200px, será verde, y si es mayor a 1200px, será coral.
